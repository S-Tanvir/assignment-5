Question-1: What are the different ways to select an element in the DOM?

There are many ways to select an element in the DOM.Five ways to select an element in the DOM are given below :

1.querySelectorAll()
2.querySelector()
3.getElementById()
4.getElementsByClassName()
5.getElementsByTagName()

Question-2: What is the difference between innerHTML, innerText, and textContent ?

innerHTML : HTML markup
innerText : Visible text after considering styles
textContent : All text content even from hidden elements

Question-3: What is event delegation in the DOM?

Event delegation is a way of handling events in JavaScript by attaching a single event listener to a parent element instead of attaching one to each child element.

Question-4: What is event bubbling in the DOM?

Event bubbling is a process in the DOM (Document Object Model) where an event that is triggered on an element "bubbles up" to its parent elements, and then continues to move upward to the root of the document.

Question-5: How do you create, add, and remove elements using JavaScript?

1.Create Elements : document.createElement()
2.Add Elements : parentElement.appendChild(newElement)
3.Remove Elements : element.remove() or parentElement.removeChild(childElement)
